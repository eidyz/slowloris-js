# SLOW LORIS CLI

## Installation
    npm i @eidyz/slowloris -g

## Usage
    Usage: slowloris [options]
    
    Options:
      -h <host>       IP Address
      -u <url>        eg: www.website.com
      -p [port]       Port <defaults to 80>
